# FutureKawa User Guide

Welcome to the **FutureKawa** user guide. FutureKawa is a web application that helps
warehouse and exploitation managers, together with headquarters, supervise the
storage of green (unroasted) coffee across **Brazil**, **Ecuador** and **Colombia**.

The application lets you:

- browse coffee **lots (batches)** in each warehouse, sorted oldest-first (FIFO);
- open a lot to read its **temperature and humidity curves**;
- watch **alerts** that flag storage conditions drifting out of range, or lots that
  have been stored for too long.

This guide is written for business users. No technical knowledge is required.

## Quick reference — ideal storage conditions

Each country has an ideal temperature and humidity band, with a tolerance around it.

| Country | Ideal temperature | Ideal humidity | Tolerance |
|---|---|---|---|
| Brazil | 29 °C | 55 % | ± 3 °C / ± 2 % |
| Ecuador | 31 °C | 60 % | ± 3 °C / ± 2 % |
| Colombia | 26 °C | 80 % | ± 3 °C / ± 2 % |

A lot is also considered **too old** once it has been in storage for more than
**365 days**.
