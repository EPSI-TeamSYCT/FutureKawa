# Comprendre les alertes

Les alertes sont la manière dont FutureKawa vous signale qu'un élément demande
votre attention. Cette page explique les deux types d'alerte et quoi faire pour
chacun.

## Où trouver les alertes

- Le **Tableau de bord** affiche les alertes les plus récentes, tous pays
  confondus.
- L'écran **Alertes** liste toutes les alertes actives. Vous pouvez le filtrer par
  pays et masquer les alertes déjà traitées.

## Les deux types d'alerte

### 1. Conditions hors plage

La température ou l'humidité d'un entrepôt a dérivé **en dehors de la plage de
tolérance du pays** et y est restée. Le café stocké sur ce site risque de perdre en
qualité.

**Quoi faire :**

1. Ouvrez l'entrepôt ou le lot pour voir la courbe et confirmer que la dérive est
   durable, et non un pic isolé (voir [Lire les graphiques](lire-les-graphiques.md)).
2. Vérifiez le stockage physique : ventilation, portes, climatisation ou
   chauffage, et toute source d'humidité.
3. Ramenez les conditions dans la plage.
4. Une fois la situation maîtrisée, **marquez l'alerte comme traitée** pour qu'elle
   quitte la liste active.

### 2. Lot trop ancien

Un lot est stocké depuis **plus de 365 jours**. Le café vert âgé perd peu à peu de
sa valeur : il doit donc être expédié en priorité.

**Quoi faire :**

1. Ouvrez la liste des **Lots** — les lots les plus anciens sont déjà en haut
   (FIFO).
2. Priorisez le lot signalé pour la prochaine expédition sortante.
3. Coordonnez-vous avec l'entrepôt et le siège si nécessaire.

## Marquer une alerte comme traitée

Une fois une alerte traitée, marquez-la comme **traitée**. Elle disparaît alors de
la liste active, ce qui garde l'écran concentré sur les problèmes qui nécessitent
encore une action. Les alertes traitées restent consultables en retirant le filtre.
