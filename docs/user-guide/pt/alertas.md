# Entender os alertas

Os alertas são a forma como o FutureKawa avisa você de que algo precisa de atenção.
Esta página explica os dois tipos de alerta e o que fazer em cada caso.

## Onde encontrar os alertas

- O **Painel** mostra os alertas mais recentes de todos os países.
- A tela **Alertas** lista todos os alertas ativos. Você pode filtrá-la por país e
  ocultar os alertas que já foram tratados.

## Os dois tipos de alerta

### 1. Condições fora da faixa

A temperatura ou a umidade de um armazém desviou para **fora da faixa de tolerância
do país** e permaneceu assim. O café armazenado nesse local está em risco de perder
qualidade.

**O que fazer:**

1. Abra o armazém ou o lote para ver a curva e confirmar que o desvio é sustentado,
   e não um pico isolado (veja [Ler os gráficos](ler-graficos.md)).
2. Verifique o armazenamento físico: ventilação, portas, ar-condicionado ou
   aquecimento, e qualquer fonte de umidade.
3. Traga as condições de volta para dentro da faixa.
4. Assim que a situação estiver sob controle, **marque o alerta como tratado** para
   que ele saia da lista de ativos.

### 2. Lote antigo demais

Um lote está armazenado há **mais de 365 dias**. O café verde mais antigo perde
valor gradualmente, então deve ser priorizado para expedição.

**O que fazer:**

1. Abra a lista de **Lotes** — os lotes mais antigos já estão no topo (FIFO).
2. Priorize o lote sinalizado para a próxima expedição de saída.
3. Coordene com o armazém e a matriz conforme necessário.

## Marcar um alerta como tratado

Assim que você tiver lidado com um alerta, marque-o como **tratado**. Ele então
desaparece da lista de ativos, o que mantém a tela focada nos problemas que ainda
precisam de ação. Os alertas tratados ainda podem ser revistos removendo o filtro.
