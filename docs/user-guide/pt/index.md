# Guia do usuário FutureKawa

Bem-vindo ao guia do usuário do **FutureKawa**. O FutureKawa é uma aplicação web
que ajuda os responsáveis por armazéns e fazendas, junto com a matriz, a
supervisionar o armazenamento do café verde (não torrado) no **Brasil**, no
**Equador** e na **Colômbia**.

A aplicação permite que você:

- percorra os **lotes** de café de cada armazém, ordenados do mais antigo para o
  mais recente (FIFO);
- abra um lote para ler suas **curvas de temperatura e umidade**;
- acompanhe os **alertas** que sinalizam condições de armazenamento fora da faixa,
  ou lotes armazenados por tempo demais.

Este guia foi escrito para usuários de negócio. Nenhum conhecimento técnico é
necessário.

## Referência rápida — condições ideais de armazenamento

Cada país tem uma faixa ideal de temperatura e umidade, com uma tolerância em torno
dela.

| País | Temperatura ideal | Umidade ideal | Tolerância |
|---|---|---|---|
| Brasil | 29 °C | 55 % | ± 3 °C / ± 2 % |
| Equador | 31 °C | 60 % | ± 3 °C / ± 2 % |
| Colômbia | 26 °C | 80 % | ± 3 °C / ± 2 % |

Um lote também é considerado **antigo demais** assim que fica armazenado por mais de
**365 dias**.
