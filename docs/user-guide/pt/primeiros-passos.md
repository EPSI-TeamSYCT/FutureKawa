# Primeiros passos

Esta página explica como abrir o FutureKawa e se orientar pelas telas principais.
Consulte o [índice do guia do usuário](index.md) para o sumário completo.

## Acessar a aplicação web

1. Abra seu navegador web (Chrome, Edge, Firefox ou Safari).
2. Acesse o endereço do FutureKawa fornecido pelo seu administrador.
3. A aplicação abre diretamente no **Painel**.

A interface funciona em um computador e se adapta a tablets. Está disponível em um
tema **claro** (padrão) e um tema **escuro** — pressione `t` a qualquer momento para
alternar.

## As telas principais

O FutureKawa é organizado em torno de uma barra lateral à esquerda. Cada entrada
abre uma das telas principais:

| Tela | O que você vê |
|---|---|
| **Painel** | Uma visão geral de todos os países: números-chave, alertas recentes e a saúde de cada armazém. |
| **Lotes** | A lista completa dos lotes de café, ordenados do mais antigo para o mais recente (FIFO). |
| **Armazéns** (Entrepôts) | Os armazéns do país selecionado, cada um com suas últimas condições medidas. |
| **Alertas** (Alertes) | A lista de alertas ativos (condições fora da faixa ou lotes antigos demais). |
| **Configurações** (Paramètres) | Preferências pessoais, como o tema. |

## Selecionar um país

No topo da aplicação você pode filtrar toda a visão por **país** (Brasil, Equador,
Colômbia) e, dentro de um país, por **armazém**. Sua escolha acompanha você conforme
navega entre as telas, de modo que você sempre permaneça focado no mesmo local.

## Encontrar algo rapidamente

- Pressione `⌘K` (Mac) ou `Ctrl+K` (Windows) para abrir a **paleta de comandos** e
  buscar por um lote, um armazém, uma página ou uma ação.
- Pressione `/` para ir direto à busca.
