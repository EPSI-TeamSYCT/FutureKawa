# Primeros pasos

Esta página explica cómo abrir FutureKawa y orientarse en las pantallas
principales. Consulte el [índice de la guía de usuario](index.md) para ver la tabla
de contenidos completa.

## Acceder a la aplicación web

1. Abra su navegador web (Chrome, Edge, Firefox o Safari).
2. Vaya a la dirección de FutureKawa que le facilitó su administrador.
3. La aplicación se abre directamente en el **Panel**.

La interfaz funciona en un computador y se adapta a las tabletas. Está disponible en
un tema **claro** (predeterminado) y un tema **oscuro** — pulse `t` en cualquier
momento para alternar entre ellos.

## Las pantallas principales

FutureKawa se organiza en torno a una barra lateral izquierda. Cada entrada abre una
de las pantallas principales:

| Pantalla | Lo que ve |
|---|---|
| **Panel** | Una vista general de todos los países: cifras clave, alertas recientes y el estado de cada almacén. |
| **Lotes** | La lista completa de lotes de café, ordenados del más antiguo al más reciente (FIFO). |
| **Almacenes** (Entrepôts) | Los almacenes del país seleccionado, cada uno con sus últimas condiciones medidas. |
| **Alertas** (Alertes) | La lista de alertas activas (condiciones fuera de rango o lotes demasiado antiguos). |
| **Ajustes** (Paramètres) | Preferencias personales, como el tema. |

## Seleccionar un país

En la parte superior de la aplicación puede filtrar toda la vista por **país**
(Brasil, Ecuador, Colombia) y, dentro de un país, por **almacén**. Su elección le
acompaña al desplazarse entre pantallas, de modo que siempre permanece centrado en
el mismo sitio.

## Encontrar algo rápidamente

- Pulse `⌘K` (Mac) o `Ctrl+K` (Windows) para abrir la **paleta de comandos** y
  buscar un lote, un almacén, una página o una acción.
- Pulse `/` para ir directamente a la búsqueda.
