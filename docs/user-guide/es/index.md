# Guía de usuario de FutureKawa

Bienvenido a la guía de usuario de **FutureKawa**. FutureKawa es una aplicación web
que ayuda a los responsables de almacén y de finca, junto con la sede central, a
supervisar el almacenamiento de café verde (sin tostar) en **Brasil**, **Ecuador**
y **Colombia**.

La aplicación le permite:

- consultar los **lotes** de café de cada almacén, ordenados del más antiguo al más
  reciente (FIFO);
- abrir un lote para leer sus **curvas de temperatura y humedad**;
- vigilar las **alertas** que señalan condiciones de almacenamiento fuera de rango,
  o lotes almacenados durante demasiado tiempo.

Esta guía está dirigida a usuarios de negocio. No se requieren conocimientos
técnicos.

## Referencia rápida — condiciones de almacenamiento ideales

Cada país tiene una banda ideal de temperatura y humedad, con una tolerancia a su
alrededor.

| País | Temperatura ideal | Humedad ideal | Tolerancia |
|---|---|---|---|
| Brasil | 29 °C | 55 % | ± 3 °C / ± 2 % |
| Ecuador | 31 °C | 60 % | ± 3 °C / ± 2 % |
| Colombia | 26 °C | 80 % | ± 3 °C / ± 2 % |

Un lote también se considera **demasiado antiguo** cuando lleva almacenado más de
**365 días**.
